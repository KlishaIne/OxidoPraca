Instrukcja uruchomienia aplikacji

Ta instrukcja pomoże Ci zainstalować wszystkie niezbędne komponenty oraz uruchomić aplikację, która pobiera treść artykułu, konwertuje go na kod HTML, generuje obrazy i tworzy podgląd artykułu.

Spis treści

  1. Wymagania systemowe
  2. Instalacja Pythona 3.13
  3. Instalacja pip
  4. Instalacja wymaganych bibliotek
  5. Konfiguracja klucza API OpenAI
  6. Uruchomienie aplikacji
  7. Dodatkowe informacje

1. Wymagania systemowe

  • System operacyjny: Windows, macOS lub Linux
  • Połączenie z internetem
  • Klucz API OpenAI z dostępem do modeli GPT-3.5/GPT-4 oraz API Obrazów

2. Instalacja Pythona 3.13

Uwaga: Upewnij się, że instalujesz Python w wersji 3.13 lub nowszej.

Windows:

  1. Przejdź na stronę Python Releases for Windows.
  2. Pobierz instalator "Windows installer (64-bit)" dla wersji Python 3.13.
  3. Uruchom pobrany instalator.
  4. Ważne: Podczas instalacji zaznacz opcję "Add Python 3.13 to PATH".
  5. Postępuj zgodnie z instrukcjami instalatora.

macOS:

  1. Przejdź na stronę Python Releases for macOS.
  2. Pobierz instalator "macOS 64-bit installer" dla wersji Python 3.13.
  3. Uruchom pobrany plik .pkg i postępuj zgodnie z instrukcjami.

Linux:

  • Możesz użyć menedżera pakietów swojej dystrybucji lub pobrać źródła z oficjalnej strony Pythona.
  • Przykład dla Ubuntu/Debiana:
sudo apt update
sudo apt install python3.13

3. Instalacja pip

pip jest narzędziem do zarządzania pakietami Pythona. W nowszych wersjach Pythona pip jest już zainstalowany.

Sprawdź, czy pip jest zainstalowany:

pip --version

Jeśli otrzymasz informację o wersji pip, możesz przejść do następnego kroku.

Instalacja pip:

Jeśli pip nie jest zainstalowany, wykonaj poniższe kroki.

Windows:

  1. Pobierz plik get-pip.py.
  2. Otwórz wiersz poleceń i uruchom:
python get-pip.py

macOS/Linux:

  1. Otwórz terminal i uruchom:
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python3 get-pip.py

4. Instalacja wymaganych bibliotek

Aplikacja wymaga dwóch bibliotek: openai i requests.

Instalacja za pomocą pip:

  1. Otwórz wiersz poleceń lub terminal.
  2. Przejdź do katalogu z aplikacją generate_article.py.
  3. Wykonaj polecenie:
pip install openai requests

5. Konfiguracja klucza API OpenAI

Aby aplikacja mogła korzystać z usług OpenAI, musisz ustawić zmienną środowiskową OPENAI_API_KEY z Twoim kluczem API.

Uzyskanie klucza API:

  1. Zarejestruj się lub zaloguj na OpenAI Platform.
  2. Przejdź do strony kluczy API.
  3. Wygeneruj nowy klucz API.

Ustawienie zmiennej środowiskowej:

Windows:

  1. Otwórz menu Start i wpisz "Zmienne środowiskowe", wybierz "Edytuj zmienne środowiskowe systemu".
  2. Kliknij "Zmienne środowiskowe".
  3. W sekcji "Zmienne użytkownika" kliknij "Nowa...".
  4. Wprowadź:
    • Nazwa zmiennej: OPENAI_API_KEY
    • Wartość zmiennej: TWÓJ_KLUCZ_API
  5. Zatwierdź zmiany.

macOS/Linux:

  1. Otwórz terminal.
  2. Edytuj plik powłoki (np. .bash_profile, .bashrc lub .zshrc):
nano ~/.bash_profile
  1. Dodaj linię:
export OPENAI_API_KEY="TWÓJ_KLUCZ_API"
  1. Zapisz plik i zamknij edytor (w nano: Ctrl + O, Enter, Ctrl + X).
  2. Załaduj zmiany:
source ~/.bash_profile

6. Uruchomienie aplikacji

Kroki:

  1. Uruchom plik Run.bat
  2. Czekaj na zakończenie działania skryptu:
    • Skrypt pobierze treść artykułu, wygeneruje kod HTML, wygeneruje obrazy i zapisze pliki.
    • W trakcie działania mogą pojawić się komunikaty informujące o postępie.
  3. Sprawdź wyniki:
    • W katalogu powinny pojawić się pliki:
      • szablon.html – pusty szablon HTML.
      • podglad.html – pełny podgląd artykułu z wygenerowanymi obrazami.
    • Folder images zawierający wygenerowane obrazy.
  4. Otwórz plik podglad.html w przeglądarce:
    • Dwukrotnie kliknij plik lub otwórz go za pomocą przeglądarki internetowej.
    • Powinieneś zobaczyć artykuł z wbudowanymi obrazami.

7. Dodatkowe informacje

Uwagi:

  • Koszty i limity:
    • Generowanie obrazów za pomocą API OpenAI może wiązać się z kosztami.
    • Upewnij się, że masz wystarczające środki na koncie OpenAI.
  • Problemy z generowaniem obrazów:
    • Jeśli obrazy nie zostaną wygenerowane, upewnij się, że masz dostęp do API Obrazów OpenAI.
    • Sprawdź, czy klucz API jest poprawnie ustawiony.
  • Zmiana ustawień skryptu:
    • Możesz edytować plik generate_article.py, aby dostosować ustawienia, takie jak rozmiar obrazów czy model AI.
